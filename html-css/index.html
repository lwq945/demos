游戏背景

游戏主要围绕三大黑暗势力张开……

  • 邪灵妖风

    大唐地界内的建邺城出现邪物“妖风”，对活人的鲜血有着强烈的渴望，肆意攻击小城的居民。

  • 战神蚩尤

    蚩尤和黄帝一场大战，黄帝设下玄黄无极阵将蚩尤困于其中。战神蚩尤脱困无望，本身的善念逐渐被恶念吞噬，化出心魔邪灵。

  • 魔将九头

    蚩尤部将九头精怪带妖风部众在尘世间散播邪恶，驱除心灵的良善，将巨大的潜力引入无边的黑暗。通过这股黑暗的力量，蚩尤的力量得到增强，逐渐摆脱符咒的禁锢，甚至可以撼动阵法的束缚！

门派简介

各门派特色介绍......

  • 大唐官府

    横扫千军意气重，破釜沉舟慨而慷。天下兵法了于胸，十方无敌勇者狂。大唐子弟曾随唐皇征战天下，立下汗马功劳，门派师父程咬金更是凌烟阁功臣榜上第一猛将。门下弟子个个骁勇威武，不仅深谙为官之道，更具文韬武略。其武功招式偏 重物理攻击，以强大的爆发力著称。

  • 方寸山

    不生不灭，全气全神。熟读黄庭神兵护法，失心离魂以符为之。灵台方寸山，斜月三星洞。方寸山弟子慎守菩提祖师“养本固元，清静无为”的教导，见性明心，调和阴阳，所以他们对符法有着异乎常人的理解能力，尤善请神降圣，驱妖伏魔。他们不但能捻诀攻击敌方，还能够书符使敌方五神昏迷，进入异常状态。

  • 狮驼岭

    魔兽九相功，神力无穷。挟兽王之威，习狮象鹏之术。狮驼岭三位大王都有拿手秘技，相传有古老魔神处传来的秘术，让这个门派有一层神秘色彩。骷髅若岭，骸骨成林，狮驼岭弟子出手狠辣，无情的行事作风令人闻风丧胆。狂暴的时候他们拥有无穷神力，给予敌人致命的袭击！

  • 普陀山

    济世救人慈悲心，净瓶甘露度群生。不二法门能证果，五行伏魔亦何难。“落伽山上慈悲主，潮音洞里观世音”。拜观音姐姐为师的普陀山弟子，修习莲华妙法，心无点尘，能够不被五行拘束，更能控制这种力量。常诵金刚经可度众生，杨柳净瓶中的甘露还有着起死回生的神奇效果。
